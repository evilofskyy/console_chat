# Чат-приложение на C++

Консольное многопользовательское приложение для обмена личными и публичными сообщениями. Данные пользователей и сообщений сохраняются в текстовые файлы.

## Особенности

- Регистрация и авторизация пользователей
- Отправка личных сообщений (адресату по логину)
- Отправка публичных сообщений (всем пользователям)
- Просмотр личной переписки и общего чата
- Очистка истории сообщений текущего пользователя
- Список зарегистрированных пользователей
- **Шаблонная функция форматирования сообщений** (`formatMessage<Format>`), поддерживающая:
  - текстовый формат (используется в стандартных меню)
  - JSON-формат (вывод в консоль по запросу)
- Надёжная обработка ввода (защита от некорректного ввода)

## Требования

- Компилятор с поддержкой **C++17** (GCC 7+, Clang 5+, MSVC 2017+)
- Стандартная библиотека C++
